Contador clásico
Un contador con controles para incrementar, decrementar y resetear. El botón '+3' debe sumar tres unidades incluso si se presiona varias veces seguido — usa actualización funcional.

Objetivos
1.
Declara un estado count que empiece en 0
2.
Botón '+' suma 1, botón '−' resta 1
3.
Botón 'reset' vuelve a 0
4.
Botón '+3' suma 3 usando setCount(c => c + 1) tres veces
5.
El display debe mostrar el valor actual centrado
